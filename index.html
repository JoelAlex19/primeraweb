HTML

HTML es un lenguaje de marcado que se utiliza para crear páginas web. Es el lenguaje de marcado estándar para la creación de documentos en la world wide web. HTML se compone de una serie de elementos que se utilizan para estructirar y presentar el contenido de una página web. Estos elementos incluyen encabezados, párrafos, enlaces, imágenes tablas y muchos otros. HTML es un lenguaje muy flexible y se puede utilizar para crear una amplia variedad de sitios web, desde simples páginas estaticas hasta aplicaciones web complejas
